## Data Storage Details

### What's Stored Locally

| Data Type | Storage Method | Encryption |
|-----------|----------------|------------|
| Journal Entries | SQLite Database | AES Encrypted |
| Daily Check-ins | SQLite Database | Plain (non-sensitive) |
| Milestones | SQLite Database | Plain |
| Meeting Logs | SQLite Database | Plain |
| Settings | SQLite Database | Plain |
| Motivation Vault | SQLite Database | Plain |
| Time Capsules | SQLite Database | AES Encrypted |
| Encryption Key | Secure Keychain | Hardware-Protected |

### Encryption Method

- Algorithm: AES-256
- Key Storage: iOS Keychain / Android Keystore (hardware-backed when available)
- Key Generation: Cryptographically secure random generation
